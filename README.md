# ReactJS, Webpack, Redux

### Features

* ReactJS 17
* Webpack 5
* Redux 5
* Axios
* Eslint
* Babel

### Structure
- **src**
  - **ajax** - calls apis by Ajax
  - **contants** - const parametters
  - **components** - Web components & UI
  - **redux** - Action & reducers
  - **index.html** - static html
  - **index.js** - entry files
  - **routes.js** - routes

### Setup

```
$yarn install
```

### Build

1. Production
```
$yarn build
```
2. Development
```
$yarn build:development
```
3. local
```
$yarn start
```
4. Fix eslint format
```
$yarn lint --fix
```
